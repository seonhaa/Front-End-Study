[1]    let originalObj = {
        name: "안유진",
        group: "아이브",
        };

        console.log(originalObj);
        console.log(cloneObj);

[3]    originalObj["group"] = "코드팩토리";

[1] 객체 하나 만들고 originalObj라는 이름표를 붙인다.
[2] cloneObj는 상자를 새로 만드는 게 아니라 같은 상자를 같이 바라보는 두 번째 이름표다.
[3] originalObj 안의 값을 바꾸면 둘 다 같은 상자를 보므로 cloneObj에서도 값이 바뀌어 있다.

* 값 자체를 복사하는 것이 아니라, 값이 저장된 메모리 주소(참조)를 복사하는 방법
-> 둘 다 같은 객체 / 배열을 가리킴
